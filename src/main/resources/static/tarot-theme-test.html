테마별 타로 콘텐츠 통합 테스트

공통 설정

예: http://localhost:8080
/api/auth/login 으로 USER 로그인 후 받은 accessToken 그대로 입력

1. 연애운 타로 </api/tarot/themes/love> POST

백엔드에는 코드(SOME/REUNION/COUPLE/SOLO/CRUSH)가 전달되고,
응답에는 situationLabel(썸/재회/...) 과 잘려진 resultText 가 내려옵니다.
요청 JSON
응답 JSON

2. 월간 테마 운세 </api/tarot/themes/monthly> POST

ThemeTarotRequest.themeId (2~7 범위)
선택된 테마에서 랜덤 카드 1장, resultText 전체를 반환합니다.
요청 JSON
응답 JSON

3. 오늘의 운세 </api/tarot/themes/daily> GET

TarotThemeService.drawDailyTarot() → themeId = 8 고정으로 랜덤 카드 1장.
요청 정보
응답 JSON
